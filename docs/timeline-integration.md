# Интеграция вкладки «Таймлайн» / «Стрела времени»

Эта инструкция описывает, как встроить интерактивный таймлайн «Стрела времени» в существующий сайт преподавателя на базе проекта **Ph**.

## Что добавлено

- **Публичная страница таймлайна** — `/timeline`
- **Админ-панель управления** — `/admin/timeline`
- **API endpoints** — `timeline.list` (public), `timeline.create/update/delete` (admin)
- **Таблица в БД** — `timeline_entries`
- **Seed-данные** — 14 готовых записей (учёные и открытия)

## Быстрый запуск

1. Примените миграции БД:
   ```bash
   cd app/
   npm run db:migrate
   ```
   > Если команда `db:migrate` не отрабатывает, выполните SQL из `app/db/migrations/0014_acoustic_magneto.sql` вручную и вставьте хеш миграции в таблицу `__drizzle_migrations`.

2. Заполните начальные данные:
   ```bash
   npx tsx db/timeline-seed.ts
   ```

3. Запустите dev-сервер:
   ```bash
   npm run dev
   ```

4. Откройте http://localhost:3000/timeline

## Права доступа

| Роль | Доступ |
|------|--------|
| Гость | Просмотр `/timeline`, фильтры, клик по блокам |
| Ученик | Просмотр `/timeline`, фильтры, клик по блокам |
| Администратор | Всё вышеперечисленное + `/admin/timeline` (CRUD) |

## Как добавить ссылку в шапку другого сайта

Если вы интегрируете таймлайн в сторонний сайт преподавателя, добавьте ссылку в навигационное меню:

```html
<a href="/timeline">Таймлайн</a>
```

Для React Router:

```jsx
import { Link } from "react-router";

<Link to="/timeline">Таймлайн</Link>
```

Для обычного HTML/CSS/JS:

```html
<nav>
  <a href="/timeline">Таймлайн</a>
</nav>
```

## API для получения данных (для стороннего фронтенда)

Если таймлайн встраивается отдельно от React-приложения, можно обращаться к tRPC endpoint:

```bash
GET /trpc/timeline.list
```

Вернёт JSON-массив записей:

```json
[
  {
    "id": 1,
    "type": "physicist",
    "name": "Альберт Эйнштейн",
    "yearStart": 1879,
    "yearEnd": 1955,
    "description": "...",
    "portraitUrl": "https://...",
    "color": "#f97316",
    "sortOrder": 5
  }
]
```

## Структура полей записи

| Поле | Тип | Описание |
|------|-----|----------|
| `type` | `physicist` / `discovery` | Тип записи |
| `name` | string | Имя учёного или название открытия |
| `yearStart` | number | Год начала / год открытия |
| `yearEnd` | number \| null | Год смерти (только для учёных) |
| `description` | string | Текст всплывающего окна |
| `portraitUrl` | string \| null | URL изображения |
| `color` | string | Цвет блока/метки (hex) |
| `sortOrder` | number | Порядок отображения |

## Управление записями

Администратор может управлять записями через `/admin/timeline`:

- Добавлять новых учёных и открытия
- Редактировать существующие записи
- Удалять записи
- Изменять цвета, порядок и изображения

Все изменения сразу отображаются на публичном таймлайне.

## Визуальные особенности

- Горизонтальная золотая стрелка с разметкой 1600–2020 (шаг 50 лет)
- Прямоугольные блоки учёных над стрелкой
- Круглые метки открытий под/на стрелке
- Фон: золотая туманность, звёздное скопление, синяя координатная сетка
- Футуристическое всплывающее окно с синей подсветкой для записей Эйнштейна / 1905 года
- Фильтры по типу и веку
- Адаптивная вёрстка для мобильных устройств
