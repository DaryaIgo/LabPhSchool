# Быстрый запуск Академии Кванта

## Требования

- Node.js 20+
- MySQL (`brew install mysql && brew services start mysql`)
- Python 3 (для скрипта `run.py`)

## Команды

```bash
# Полная настройка (зависимости, миграции, сид)
python3 run.py setup

# Запуск dev-сервера
python3 run.py dev

# Остановить сервер (из другого терминала)
python3 run.py stop

# Перезапустить сервер
python3 run.py restart

# Или всё сразу: setup + dev
python3 run.py full

# Сборка и запуск production
python3 run.py full --production
```

Сайт откроется на **http://localhost:3000**.

## Ручной запуск

Если предпочитаете без Python:

```bash
cd app/
npm install
npm run db:migrate
npx tsx db/seed.ts
npm run dev
```
